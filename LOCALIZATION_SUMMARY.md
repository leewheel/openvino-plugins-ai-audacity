## 🚀 使用方法

### 对于最终用户

1. **下载安装程序**
   ```
   https://github.com/intel/openvino-plugins-ai-audacity/releases
   ```

2. **选择中文界面**
   - 运行安装程序
   - 选择"简体中文"
   - 按照中文提示完成安装

3. **查看中文文档**
   - 阅读 `README_CN.md` 了解项目
   - 查看 `doc/中文用户指南.md` 学习使用

### 对于开发者

1. **启用界面汉化**（需要重新编译）
   ```bash
   cmake -DENABLE_CHINESE_LOCALIZATION=ON ..
   make
   ```

2. **集成翻译到代码**
   ```cpp
   #include "OVLocalization.h"
   
   // 使用本地化宏
   XO("Text") → OV_XO("Text")
   XXO("Text") → OV_XXO("Text")
   ```

3. **添加新翻译**
   - 编辑 `mod-openvino/OVStringsChinese.h`
   - 添加翻译条目
   - 重新编译
