## 🚀 如何使用汉化版本

### 方法 1: 使用安装程序（推荐）

1. 下载安装程序
2. 运行时选择"简体中文"语言
3. 安装程序界面将显示中文
4. 按照中文提示完成安装

### 方法 2: 启用插件界面中文（需要重新编译）

要启用插件界面的中文显示，需要：

1. **修改 CMakeLists.txt**
   ```cmake
   # 在 mod-openvino/CMakeLists.txt 中添加
   add_definitions(-DENABLE_CHINESE_LOCALIZATION=1)
   ```

2. **修改源代码**
   
   在每个 `.cpp` 文件开头添加：
   ```cpp
   #include "OVLocalization.h"
   ```
   
   将字符串宏替换为本地化版本：
   ```cpp
   // 原来的代码
   XO("OpenVINO Music Separation")
   
   // 改为
   OV_XO("OpenVINO Music Separation")
   ```

3. **重新编译**
   ```bash
   cd tools/windows
   fetch_build_package.bat
   ```

## 🔍 翻译质量保证

### 术语一致性

所有技术术语都经过统一翻译：

| 英文 | 中文 | 使用场景 |
|-----|------|---------|
| Device | 设备 | 硬件设备选择 |
| Model | 模型 | AI 模型选择 |
| Inference | 推理 | AI 推理过程 |
| Encoder | 编码器 | 编码组件 |
| Decoder | 解码器 | 解码组件 |
| Stem | 轨 | 音轨分离 |
| Chunk | 块 | 数据块大小 |
| Seed | 随机种子 | 随机数生成 |
| Guidance Scale | 引导比例 | AI 生成参数 |
| Prompt | 提示词 | 文本输入 |

### 翻译原则

1. **准确性**: 技术术语翻译准确，不产生歧义
2. **一致性**: 相同术语在所有位置使用相同翻译
3. **简洁性**: 界面文本简洁明了，易于理解
4. **本地化**: 符合中文用户的使用习惯
