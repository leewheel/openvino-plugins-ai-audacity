## 🎯 快速开始

### 用户：使用中文安装程序

```bash
1. 下载安装程序
2. 运行时选择"简体中文"
3. 按提示完成安装
4. 查看 doc/中文用户指南.md
```

### 开发者：启用界面汉化

```bash
# 1. 修改 CMakeLists.txt
add_definitions(-DENABLE_CHINESE_LOCALIZATION=1)

# 2. 在源文件中
#include "OVLocalization.h"

# 3. 替换字符串宏
XO("Text") → OV_XO("Text")

# 4. 编译
cmake -DENABLE_CHINESE_LOCALIZATION=ON ..
make
```

## 💻 代码示例

### 基本用法

```cpp
// 包含头文件
#include "OVLocalization.h"

// 使用宏
const ComponentInterfaceSymbol Symbol
{ OV_XO("OpenVINO Music Separation") };

// 翻译函数
std::string translated = OVLocalization::Translate("Device");

// 检查是否启用
if (OVLocalization::IsLocalizationEnabled()) {
    // 使用中文
}
```

### 界面元素

```cpp
// 按钮
S.AddButton(OV_XO("Device Details..."));

// 选择框
.AddChoice(OV_XXO("OpenVINO Inference Device:"), ...);

// 文本框
.AddTextBox(OV_XXO("Prompt:"), ...);

// 复选框
.AddCheckBox(OV_XXO("&Advanced Options"), ...);
```

### 消息框

```cpp
// 错误消息
EffectUIServices::DoMessageBox(*this,
   OV_XO("Music Separation failed. See details in Help->Diagnostics->Show Log..."),
   wxICON_STOP,
   OV_XO("Error"));

// 信息消息
EffectUIServices::DoMessageBox(*this,
   TranslatableString(device_info, {}),
   wxICON_INFORMATION,
   OV_XO("OpenVINO Device Details"));
```

### 进度消息

```cpp
// 简单进度
TotalProgress(0.01, OV_XO("Compiling AI Model..."));

// 动态消息
std::string msg = OVLocalization::Translate("Loading") + " " + model_name;
TotalProgress(0.5, TranslatableString{ wxString(msg), {} });
```

## 🔧 添加新翻译

### 1. 编辑 OVStringsChinese.h

```cpp
// 在相应的 map 中添加
inline const std::map<std::string, std::string> YourModuleStrings = {
    {"English Text", "中文文本"},
    {"Another Text", "另一个文本"}
};

// 在 GetTranslation 函数中添加查找
if (YourModuleStrings.count(key)) return YourModuleStrings.at(key);
```

### 2. 在代码中使用

```cpp
#include "OVLocalization.h"

// 使用宏
OV_XO("English Text")  // 自动翻译为 "中文文本"

// 或使用函数
OVLocalization::Translate("English Text")  // 返回 "中文文本"
```

## 🐛 常见问题

### Q: 翻译没有生效？

```bash
# 检查编译选项
cmake -DENABLE_CHINESE_LOCALIZATION=ON ..

# 检查是否包含头文件
#include "OVLocalization.h"

# 检查是否使用了正确的宏
OV_XO("Text")  // ✅ 正确
XO("Text")     // ❌ 错误
```

### Q: 如何添加新语言？

```cpp
// 1. 创建新的翻译文件
OVStringsJapanese.h

// 2. 修改 OVLocalization.h
#ifdef ENABLE_JAPANESE_LOCALIZATION
    #include "OVStringsJapanese.h"
#endif

// 3. 添加编译选项
-DENABLE_JAPANESE_LOCALIZATION=1
```

### Q: 如何测试翻译？

```bash
# 1. 启用汉化编译
cmake -DENABLE_CHINESE_LOCALIZATION=ON ..

# 2. 编译
make

# 3. 安装并测试
# 检查所有界面元素是否显示中文
```
