# OpenVINO AI Plugins for Audacity 中文用户指南

## 欢迎使用

欢迎使用 OpenVINO™ AI Plugins for Audacity！这是一套为 Audacity® 音频编辑软件提供的 AI 功能插件，所有 AI 功能都在您的电脑上本地运行，无需互联网连接。

## 功能介绍

### 1. 🎵 音乐分离

将单声道或立体声音轨分离成独立的音轨：

- **2轨模式**: 伴奏 + 人声
- **4轨模式**: 鼓 + 贝斯 + 人声 + 其他乐器

**使用方法**:
1. 选择一段音频
2. 点击菜单：效果 → OpenVINO Music Separation
3. 选择分离模式和推理设备
4. 点击"应用"

### 2. 🧹 噪声抑制

从语音音频中去除背景噪声。

**支持的模型**:
- DeepFilterNet2 (推荐)
- DeepFilterNet3 (推荐)
- DenseUNet (传统模型)

**使用方法**:
1. 选择包含语音的音频片段
2. 点击菜单：效果 → OpenVINO Noise Suppression
3. 选择噪声抑制模型
4. 调整衰减限制（可选）
5. 点击"应用"

### 3. 🎼 音乐生成与续写

使用 MusicGen LLM 根据文本提示生成音乐片段。

**功能特点**:
- 文本转音乐
- 音乐续写
- 支持单声道和立体声
- 可调节生成参数

**使用方法**:
1. 点击菜单：生成 → OpenVINO Music Generation
2. 输入音乐描述（提示词）
3. 设置时长和其他参数
4. 选择模型和设备
5. 点击"生成"

**提示词示例**:
- "轻快的钢琴曲，适合早晨"
- "激昂的摇滚吉他独奏"
- "平静的环境音乐，带有海浪声"

### 4. 🎤 Whisper 语音转录

将语音转录为文本标签轨道。

**功能特点**:
- 支持多语言识别（包括中文）
- 自动语言检测
- 翻译功能（翻译为英文）
- 多种模型大小可选

**使用方法**:
1. 选择包含语音的音频
2. 点击菜单：分析 → OpenVINO Whisper Transcription
3. 选择 Whisper 模型
4. 选择模式（转录/翻译）
5. 选择源语言（或选择"自动"）
6. 点击"应用"

**模型选择建议**:
- **base**: 最快，适合一般使用
- **small**: 平衡速度和准确度
- **medium**: 更高准确度
- **large**: 最高准确度，处理时间较长

### 5. ✨ 超分辨率

将音频升频到 48kHz，提升音质。

**模型类型**:
- **基础（通用）**: 适合所有类型的音频
- **语音**: 专门优化语音音频

**使用方法**:
1. 选择要增强的音频
2. 点击菜单：效果 → OpenVINO Super Resolution
3. 选择模型类型
4. 配置设备和参数
5. 点击"应用"

## 安装指南

### 系统要求

- Windows 10/11 (64位)
- Audacity 3.4.0 或更高版本
- 至少 8GB RAM（推荐 16GB）
- 支持的硬件加速器：CPU、GPU、NPU

### 安装步骤

1. **下载安装程序**
   - 访问 [GitHub Releases](https://github.com/intel/openvino-plugins-ai-audacity/releases)
   - 下载最新的 Windows 安装程序

2. **运行安装程序**
   - 双击安装程序
   - 选择语言：**简体中文**
   - 按照向导完成安装

3. **选择要安装的模型**
   
   安装程序会提示选择要下载的 AI 模型：
   
   **推荐安装**（适合大多数用户）:
   - ✅ 音乐分离模型
   - ✅ 超分辨率模型（基础 + 语音）
   - ✅ 噪声抑制（DeepFilterNet2 + DeepFilterNet3）
   - ✅ Whisper Base 模型
   - ✅ 音乐生成模型（小型单声道）
   
   **完整安装**（需要更多磁盘空间）:
   - 包含所有模型和所有 Whisper 模型大小

4. **启用插件**
   
   安装完成后：
   - 打开 Audacity
   - 进入：编辑 → 首选项 → 模块
   - 找到 "mod-openvino"
   - 设置为 "已启用"
   - 关闭并重新打开 Audacity

5. **验证安装**
   
   检查菜单中是否出现 OpenVINO 相关选项：
   - 效果菜单中的 OpenVINO 选项
   - 生成菜单中的 OpenVINO Music Generation
   - 分析菜单中的 OpenVINO Whisper Transcription

## 设备选择指南

### CPU
- **优点**: 兼容性最好，所有功能都支持
- **缺点**: 处理速度较慢
- **适用**: 没有独立显卡的用户

### GPU
- **优点**: 处理速度快，适合大多数任务
- **缺点**: 需要支持的显卡
- **适用**: 有 Intel 或 NVIDIA 显卡的用户

### NPU (神经处理单元)
- **优点**: 专为 AI 设计，功耗低
- **缺点**: 仅部分新型处理器支持，功能有限
- **适用**: Intel Core Ultra 处理器用户

### 设备选择建议

| 功能 | 推荐设备 | 备选设备 |
|-----|---------|---------|
| 音乐分离 | GPU | CPU |
| 噪声抑制 | GPU / NPU | CPU |
| 音乐生成 | GPU | CPU |
| Whisper 转录 | GPU / NPU | CPU |
| 超分辨率 | GPU | CPU |

## 常见问题

### Q: 首次运行为什么很慢？

A: 首次在某个设备上运行功能时，OpenVINO 需要为该设备编译 AI 模型。这个过程可能需要 10-30 秒。编译后的模型会被缓存，后续运行会快很多。

### Q: 如何提高处理速度？

A: 
1. 使用 GPU 而不是 CPU
2. 对于音乐生成，选择 int8 模型而不是 fp16
3. 对于 Whisper，使用较小的模型（如 base 或 small）
4. 关闭不必要的高级选项

### Q: 模型存储在哪里？

A: 
- 模型文件：`<Audacity安装目录>\openvino-models\`
- 编译缓存：`C:\Users\<用户名>\AppData\Roaming\audacity\openvino-model-cache\`

### Q: 如何卸载？

A:
1. 打开 Windows 设置
2. 进入"应用"
3. 找到 "OpenVINO AI Plugins for Audacity"
4. 点击"卸载"

### Q: 支持哪些音频格式？

A: 支持 Audacity 支持的所有音频格式，包括：
- WAV
- MP3
- FLAC
- OGG
- M4A/AAC

### Q: 可以离线使用吗？

A: 是的！所有功能都在本地运行，完全不需要互联网连接。

### Q: 音乐生成的版权归谁？

A: 使用 MusicGen 生成的音乐，版权归您所有。但请注意 MusicGen 模型本身的许可证条款。

### Q: Whisper 转录准确度如何？

A: 
- 对于清晰的语音，准确度很高（95%+）
- 有背景噪音时，建议先使用噪声抑制
- 使用更大的模型（medium/large）可以提高准确度
- 中文识别效果良好

## 性能优化建议

### 1. 硬件优化

- **升级驱动**: 确保显卡驱动是最新版本
- **内存**: 至少 16GB RAM 用于处理大文件
- **存储**: 使用 SSD 可以加快模型加载速度

### 2. 软件设置

- **音乐分离**: 
  - 减少 Shifts 数量（默认为 1）
  - 使用 2轨模式而不是 4轨模式

- **音乐生成**:
  - 先生成短片段（5秒）测试效果
  - 使用 int8 模型
  - 降低 TopK 值

- **超分辨率**:
  - 使用 10.24 秒块大小
  - 减少 Steps 数量（默认 50）

### 3. 工作流程优化

1. **批量处理**: 一次处理多个相似任务
2. **预加载模型**: 使用"卸载模型"按钮管理内存
3. **分段处理**: 将长音频分成小段处理

## 使用技巧

### 音乐分离技巧

- 处理前先归一化音频
- 对于低质量音频，结果可能不理想
- 可以多次运行以获得更好的分离效果

### 噪声抑制技巧

- 衰减限制设置：
  - 轻度降噪：6-12 dB
  - 中度降噪：18-24 dB
  - 完全降噪：100 dB
- 对于音乐，使用较低的衰减限制
- 对于语音，可以使用较高的衰减限制

### 音乐生成技巧

- **好的提示词**:
  - 具体描述风格、乐器、情绪
  - 例如："欢快的爵士钢琴，带有轻快的鼓点"
  
- **使用种子**:
  - 设置固定种子可以重现相同的结果
  - 实验时留空，让系统随机生成

- **音频续写**:
  - 选择现有音频的末尾
  - 系统会自动检测并启用续写模式

### Whisper 转录技巧

- 对于中文语音，选择"chinese"作为源语言
- 使用"初始提示"提供上下文（如专有名词）
- 对于长音频，系统会自动分段处理

## 故障排除

### 问题：插件没有出现在菜单中

**解决方案**:
1. 检查插件是否已启用（首选项 → 模块）
2. 确认 Audacity 版本 ≥ 3.4.0
3. 重新安装插件

### 问题：加载模型失败

**解决方案**:
1. 检查模型文件是否完整
2. 确认有足够的磁盘空间
3. 检查防病毒软件是否阻止
4. 查看日志：帮助 → 诊断 → 显示日志

### 问题：处理时崩溃

**解决方案**:
1. 更新显卡驱动
2. 尝试使用 CPU 设备
3. 减小处理的音频长度
4. 增加系统内存

### 问题：结果质量不佳

**解决方案**:
- **音乐分离**: 增加 Shifts 数量
- **噪声抑制**: 尝试不同的模型
- **音乐生成**: 调整 Guidance Scale 和 TopK
- **Whisper**: 使用更大的模型
- **超分辨率**: 增加 Steps 数量
