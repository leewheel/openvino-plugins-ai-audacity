# 汉化集成示例

本文档展示如何将中文翻译集成到现有代码中。

## 示例 1: 音乐分离模块

### 原始代码 (OVMusicSeparation.cpp)

```cpp
const ComponentInterfaceSymbol EffectOVMusicSeparation::Symbol
{ XO("OpenVINO Music Separation") };

TranslatableString EffectOVMusicSeparation::GetDescription() const
{
   return XO("Splits a stereo track into 4 new tracks -- Bass, Drums, Vocals, Others");
}
```

### 汉化后的代码

```cpp
#include "OVLocalization.h"

const ComponentInterfaceSymbol EffectOVMusicSeparation::Symbol
{ OV_XO("OpenVINO Music Separation") };

TranslatableString EffectOVMusicSeparation::GetDescription() const
{
   return OV_XO("Splits a stereo track into 4 new tracks -- Bass, Drums, Vocals, Others");
}
```

## 示例 6: Whisper 语言列表

### 原始代码

```cpp
static const lang_entry g_lang_entries[] = {
    { "chinese", "zh", 1, },
    { "english", "en", 0, },
    // ... 更多语言
};

for (auto e : g_lang_entries)
{
   mSupportedLanguages.push_back(e.lang_full_string);
}

for (auto l : mSupportedLanguages)
{
   mGuiLanguageSelections.push_back({ TranslatableString{ wxString(l), {}} });

```

### 汉化后的代码

```cpp
#include "OVLocalization.h"

static const lang_entry g_lang_entries[] = {
    { "chinese", "zh", 1, },
    { "english", "en", 0, },
    // ... 更多语言
};

for (auto e : g_lang_entries)
{
   mSupportedLanguages.push_back(e.lang_full_string);
}

for (auto l : mSupportedLanguages)
{
   std::string translated = OVLocalization::Translate(l);
   mGuiLanguageSelections.push_back({ TranslatableString{ wxString(translated), {}} });
}
```

## 示例 8: 超分辨率模型选择

### 原始代码

```cpp
#define BASIC_MODEL_STRING "Basic (General)"
#define SPEECH_MODEL_STRING "Speech"

// 在代码中使用
if (chosen_model == BASIC_MODEL_STRING)
{
   config.model_selection = ov_audiosr::AudioSRModel::BASIC;
}
else if (chosen_model == SPEECH_MODEL_STRING)
{
   config.model_selection = ov_audiosr::AudioSRModel::SPEECH;
```

### 汉化后的代码

```cpp

#define BASIC_MODEL_STRING "Basic (General)"
#define SPEECH_MODEL_STRING "Speech"

// 在 GUI 中显示翻译后的文本
mGuiModelSelections.push_back({ 
   TranslatableString{ 
      wxString(OVLocalization::Translate(BASIC_MODEL_STRING)), 
      {} 
   } 
});

// 在代码中仍使用英文常量进行比较
if (chosen_model == BASIC_MODEL_STRING)
{
   config.model_selection = ov_audiosr::AudioSRModel::BASIC;
}
else if (chosen_model == SPEECH_MODEL_STRING)
{
   config.model_selection = ov_audiosr::AudioSRModel::SPEECH;
}
```

## 编译选项

### 启用中文本地化

```bash
# Windows
cmake -DENABLE_CHINESE_LOCALIZATION=ON ..

# 或在 CMake GUI 中勾选 ENABLE_CHINESE_LOCALIZATION 选项
```

### 禁用中文本地化（默认）

```bash
cmake -DENABLE_CHINESE_LOCALIZATION=OFF ..
# 或者不指定该选项
```

## 常见问题

### Q: 为什么有些文本没有翻译？

A: 检查以下几点：
1. 是否在编译时启用了 `ENABLE_CHINESE_LOCALIZATION`
2. 是否在源文件中包含了 `OVLocalization.h`
3. 是否使用了 `OV_XO()` 或 `OV_XXO()` 宏
4. 翻译字符串是否在 `OVStringsChinese.h` 中定义

### Q: 如何添加新的翻译？

A: 
1. 在 `OVStringsChinese.h` 中添加翻译条目
2. 在源代码中使用 `OV_XO()` 或 `OVLocalization::Translate()`
3. 重新编译

### Q: 可以在运行时切换语言吗？

A: 当前版本需要重新编译。未来版本将支持运行时切换。
