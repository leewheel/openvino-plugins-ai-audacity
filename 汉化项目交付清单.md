   ```bash

   make
   ```

## ✨ 技术亮点

### 1. 灵活的本地化框架

```cpp
// 编译时控制
#define ENABLE_CHINESE_LOCALIZATION 1

// 便捷的宏
OV_XO("Text")   // 自动翻译
OV_XXO("Text")  // 自动翻译

// 运行时翻译
OVLocalization::Translate("Text")
```
